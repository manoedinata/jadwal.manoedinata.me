RIP Website Jadwal...

Disinilah tempat website jadwal SMAN 1 Boyolangu buatan @manoedinata.
Namun saat ini sudah tidak aktif lagi...

Website Jadwal, Ndra (https://jadwal.manoedinata.me) sudah tidak aktif lagi. Jika kamu ingin melihat jadwal, mohon maaf kamu harus melihat dokumen jadwal secara manual, atau meminta jadwal dari teman. Jika sudah di-screenshot ya alhamdulillah.

Website ini dihentikan karena kurangnya biaya maintenance server, disebabkan oleh tidak adanya 'uang kopi' sebagai pendukung layanan ini. Perlu diketahui bahwa pengelolaan data dan server memakan biaya sebesar Rp. 50.000 / bulan. Bagi saya itu nominal yang agak besar. Oh ya, jangan bully saya. Saya nabung kok.

Ditambah lagi dengan adanya masalah dari pihak penyedia server yang menyebabkan seluruh server mati, termasuk server saya dan Robotika Smaboy, semangatku untuk mengurus ulang layanan agak berkurang.
